Plain HTML host page

This is a framework-free page — no React, no Tailwind. The floating assistant in the corner is injected entirely by widget.js, isolated in a Shadow DOM. This is exactly how it embeds into your website.

The whole integration is the two tags below:

<script src="/widget.js" defer></script>
<div id="voice-ai-widget" data-api="http://localhost:8000"
     data-title="Company Assistant"></div>
